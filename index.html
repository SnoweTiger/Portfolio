The API Mystery game

The API Mystery is an exciting detective game in which you will solve crimes and practice working with Web APIs, RPC and GraphQL.
In the game, you will use these technologies to interact with various services and programs and to get the information you need for your investigation. You will send requests to the Web API, use RPC to call procedures remotely, and work with GraphQL to retrieve data.

Sale bot Bella

This is salebot for recived orders. It's work with Telegramme (API) or any other messenger (Selenium + WebClient). I called it Bella, because it was written for a company that sells Italian stone products. bella parses the question received from the client. Looking for intent in it. If the intent is an order, checks if there is enough information for the order. If not, then asks clarifying questions.

DrFreidBOT

The bot determines if the given question is among the examples for intents. If there is, then it answers with a phrase for this intent. If not, generate an anwser by generative neural network, trained on the phrases of Sigmund Freud. Currently, only russian language is supported, but you can easily retrain the model using english or german phrases.
