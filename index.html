Sale bot Bella

This is Telegramme and WhatsApp bot for recived orders. I called it Bella, because it was written for a company that sells Italian stone products. bella parses the question received from the client. Looking for intent in it. If the intent is an order, checks if there is enough information for the order. If not, then asks clarifying questions.

DrFreidBOT

The bot determines if the given question is among the examples for intents. If there is, then it answers with a phrase for this intent. If not, generate an anwser by generative neural network, trained on the phrases of Sigmund Freud. Currently, only russian language is supported, but you can easily retrain the model using english or german phrases.
