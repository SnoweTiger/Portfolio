The API Mystery game

The API Mystery - это увлекательная детективная игра, в которой вам предстоит раскрыть преступления и тренироваться в работе с Web API, RPC и GraphQL.
В игре вы будете использовать эти технологии для взаимодействия с различными сервисами и программами, а также для получения необходимой информации для расследования. Вы будете отправлять запросы к Web API, использовать RPC для удаленного вызова процедур и работать с GraphQL для получения данных.

К репозиторию APIMystery

StoneStockSystem (FastAPI/React/Flutter)

StoneStockSystem - система управления запасами и торговлей для компаний, торгующих камнем. Традиционно состоит из 3 частей: сервер на FastAPI, веб-приложение на ReactJS и мобильное приложение на Flutter.

Конвертор кошек в тигров (CycleGAN)

Я собрал датасет из фото кошек и тигров с BING Images. Генератор состоит из 3-х сверточных и 3-х residual блоков. Сеть обучалась 30 эпох до получения достаточного результата.

Также в репозитории есть скрипт для пропуска видео или картинок через нейросеть. Источником видео также может быть веб-камера.

К репозиторию Cat2Tiger

Торговый чат-бот Bella

Данный чат-бот предназначен для приема заказов. Работа с Telegramme (API) или любым другим мессенджером (Selenium + WebClient). Я назвал его Белла, так как он был написана для компании, торгующей итальянскими изделиями из камня. Bella анализирует вопрос, полученный от клиента. Ищит в нем интент. Если целью является заказ, проверяет, достаточно ли информации для заказа. Если нет, то задает необходимые уточняющие вопросы.

DrFreidBOT

Бот определяет, входит ли заданный вопрос в число предопределенных интентов. Если есть, то отвечает случайной из заданных фраз. Если нет, формирует ответ генеративной нейронной сетью, обученной на фразах Зигмунда Фрейда. Получается достаточно правдопобный ответ с стиле доктора. В настоящее время поддерживается только русский язык, но вы можете легко переобучить модель, используя английские или немецкие фразы.
